RUS:

Программа "Davarm"(Davarm.py) 2D аркада с реалистичной графикой.
Цель игры: убить как можно больше зомби или убить босса.

Клавиши 1, 2, 3, 4, 5 позволяет поменять музыку на понравивашуюся
Клавиши для насыщения игры, при которых программа выдает то или иное действие: F, O
Нажатие на одну из белых полос дороги - включает секретную музыку
Выход за пределы карты с помощью секретного прохода (осторожно! скример)
Сочетание клавиш:
F + B - появление light босса игры
F + H + B - появление hard босса игры
Если босс не спавнится, то встаньте на один из краёв карты.

ENG:

The program "Davarm"(Davarm.py ) 2D arcade game with realistic graphics.
The goal of the game is to kill as many zombies as possible or kill the boss.

Keys 1, 2, 3, 4, 5 allows you to change the music to your favorite
The keys for saturation of the game, in which the program outputs this or that action: F, O
Clicking on one of the white lanes of the road - turns on secret music
Going outside the map using a secret passageway (caution! screamer)
Keyboard shortcut:
F + B - the appearance of the light boss of the game
F + H + B - appearance of the game's hard boss
If the boss does not match, then stand on one of the edges of the map.
